05 Sobre el proyecto
Contexto

Proyecto personal desarrollado desde cero para aprender a construir agentes de IA aplicados a ciberseguridad. Incluye validaciones de seguridad, tests automatizados y pipeline CI/CD completo con GitHub Actions.

Stack tecnológico
Python 3.12 Claude API ReAct pattern VirusTotal API AbuseIPDB API GitHub Actions unittest requests python-dotenv
Seguridad del código

Sanitización de inputs contra SQL injection, XSS y command injection. Validación estricta de URLs antes de procesarlas. Las API keys nunca se exponen en logs ni resultados. Límite de 10 llamadas por análisis.
